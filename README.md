# TIG Stack with Mosquitto

Shell Scripts to setup __Telegraf__, __InfluxDB__, __Grafana__ on Systems with Linux Distributions.


## Requirements

- Scripts should be run with __root__ privileges


## Versions

| OSS  | Version |
| ---  | ------- |
| `influxdb` | `v1.8.0` |
| `telegraf` | `v1.14.5`|
| `grafana`  | `v7.0.5` |
